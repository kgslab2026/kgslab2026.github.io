우리집 텃밭 — 개인정보처리방침

최종 업데이트: 2026년 5월 5일

한 줄 요약 우리집 텃밭은 텃밭 기록과 사진을 사용자의 기기에 저장하며, 날씨/지도/분양 정보 조회를 위해 외부 서비스(예: 카카오맵) 및 공공 API와 통신할 수 있습니다.

1. 수집하는 정보

본 앱은 다음의 데이터만 사용합니다.

  • 텃밭 구성 정보 (심은 채소, 심은 날짜, 수확 예정일, 물준 날짜)
  • 성장 기록 (메모, 사진, 기록 일시)
  • 사용자 정의 작물 정보
  • 날씨 조회를 위한 현재 위치 (앱 실행 중 일시적으로만 사용)
위 데이터(텃밭 구성, 성장 기록, 사용자 정의 작물)는 기기 내 로컬 저장소(localStorage)에 저장됩니다. 위치 정보는 날씨 정보 표시 목적으로만 사용되며, 앱 내부에 별도로 저장하지 않습니다.

2. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 기기 고유 식별자
  • 광고 식별자
  • 촬영한 사진의 위치 메타데이터 (EXIF)

3. 권한 사용 목적

  • 위치 권한 — 현재 위치 기반 날씨 정보 표시에만 사용. 위치 데이터는 저장하지 않습니다.
  • 카메라 권한 — 성장 기록에 사진을 직접 촬영하거나 갤러리에서 선택할 때 사용.
  • 알림 권한 — 수확 예정일 7일 전·1일 전 로컬 알림 발송. 외부 서버를 사용하지 않는 기기 내 알림입니다.

4. 데이터 저장 및 삭제

모든 기록은 사용자의 기기에만 저장됩니다. 다음 방법으로 언제든지 삭제할 수 있습니다.

  • 앱 내 "전체 비우기" 버튼으로 텃밭 데이터 삭제
  • 성장 기록 개별 삭제
  • Android 설정 → 앱 → 우리집 텃밭 → 데이터 삭제

5. 외부 서비스

본 앱은 다음 외부 서비스를 사용하며, 기능 제공을 위해 필요한 범위에서 네트워크 통신이 발생할 수 있습니다.

  • 카카오맵 API — 텃밭 분양 정보 지도 표시에 사용됩니다. 카카오 개인정보처리방침: kakao.com/policy/privacy
  • 기상청 날씨 API — 현재 위치의 날씨 정보를 불러오는 데 사용됩니다.
  • 농사로 API (농림수산식품교육문화정보원) — 텃밭 분양 정보를 불러오는 데 사용됩니다.

본 앱은 광고 식별자 기반 광고 SDK 및 사용자 행동 분석 SDK를 사용하지 않습니다.

6. 아동 개인정보 보호

본 앱은 13세 미만 아동을 대상으로 개인정보를 수집하지 않습니다. 어떠한 개인정보도 수집하지 않으므로 아동이 안전하게 이용할 수 있습니다.

7. 방침 변경

개인정보처리방침이 변경될 경우 본 페이지에서 공지하며, 상단의 최종 업데이트 날짜를 변경합니다.
