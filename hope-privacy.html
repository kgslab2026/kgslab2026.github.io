개인정보 처리방침

실종아동찾기-Hope · 최종 업데이트: 2026년 5월 11일
실종아동찾기-Hope는 사용자의 개인정보를 수집하거나 외부 서버로 전송하지 않습니다.
앱은 경찰청 안전Dream 공개 API의 실종아동 정보를 조회하는 용도로만 사용됩니다.

1. 수집하는 정보

본 앱은 사용자의 개인정보를 수집하지 않습니다. 앱 사용 시 별도의 회원가입, 로그인, 개인 식별 정보 입력이 필요하지 않습니다.

2. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 광고 식별자, 기기 고유 식별자
  • 위치 정보
  • 검색 기록 및 사용 패턴

3. 외부 API 사용

본 앱은 경찰청이 운영하는 안전Dream OPEN API (safe182.go.kr)를 통해 공개된 실종아동 정보를 조회합니다.

  • 조회되는 정보는 경찰청이 공개한 실종아동 관련 공공데이터입니다.
  • 사용자의 개인정보는 해당 API로 전송되지 않습니다.
  • API 이용 시 검색 조건(지역, 이름 등)만 전달되며, 이는 사용자 기기에 저장되지 않습니다.

4. 권한 사용 목적

  • 인터넷 권한: 경찰청 안전Dream API를 통해 실종아동 정보를 조회하는 데 사용됩니다.
  • 네트워크 상태 확인 권한: 네트워크 연결 여부를 확인하여 오류 안내에 사용됩니다.
  • 전화 권한 (CALL_PHONE): 실종아동 신고 버튼 클릭 시 182 신고 전화 연결에만 사용됩니다.

5. 데이터 저장

본 앱은 사용자 데이터를 기기 내부 또는 외부 서버에 저장하지 않습니다. 앱 종료 시 모든 조회 데이터는 메모리에서 제거됩니다.

6. 제3자 제공

본 앱은 개인정보를 제3자에게 제공하지 않으며, 광고 SDK 또는 분석 SDK를 포함하지 않습니다.

7. 아동의 개인정보

본 앱은 사용자 개인정보를 수집하지 않으므로 아동 사용자에게도 동일한 정책이 적용됩니다.

8. 방침 변경

개인정보 처리방침이 변경될 경우 본 페이지의 "최종 업데이트" 날짜를 갱신하여 안내합니다.
